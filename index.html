EVE systems explorer

A very crude and basic page that allows to explore EVE online systems filtering them by "perks".

Filtering by "perks"

If you ever wanted to know which hi-sec systems have more than 6 asteroid belts or what 0.0 system has only one gate, this page can help answer that

The User Interface is very rudimentary right now, and will probably be improved in next versions.

Same goes for the backend code I used to generate the whole thing. Can really use some refactoring.

EVE University Wars

This will keep track of current Unistas deaths and kills from alliances we are at war with E-UNI/IVY league.

Hopefully, this will make it easier for Unistas to fly safer.

Why EVE University Wars?

Two reasons: first, I still see many newbros dying at the hands of war targets in hisec, losing their hard hearned stuff. Second, I wanted to (re)learn EVE API and have an excuse to mess around with Python :)

Please note

EVE University Wars cannot keep track of War Targets movements, it can only congregate public information available to make it easier to "read" what's happening in space. As usual, if you're moving stuff, please use an hauler alt and remember to plex tank it's ship.
